The Honest Caveats

1. This tests architecture, not production performance

We used simplified test scenarios. Real-world performance depends on embedding model quality, question complexity, and domain knowledge.

2. The baseline is "early memories" not "no memory"

We compared 3 memories vs 10 memories. We didn't test against random retrieval or simple keyword matching. But the question "does MORE context help?" is still valid and interesting.

3. Small but sufficient sample

n=12 stories is enough for significance (we got p=0.005), but n=30+ would be even stronger. Each story had only 10 visits.
